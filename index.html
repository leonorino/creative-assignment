Соревновательное программирование

С программированием меня познакомил мой папа, а также школьные олимпиады по информатике. Примерно 6 лет назад я поучаствовал в первой из них. Язык, который я тогда применял - C++. Учитывая мой малый на то время возраст, сложность языка программирования, отсутствие углубленных познаний в математике, шансов на успех практически не было. По моему мнению, олимпиадные задачи не имеют практического применения и мало применяются в реальных задачах.

Поэтому после 10 класса я решил изучить другие области программирования. Тем не менее, олимпиады стали моей отправной точкой, и благодаря им я знаком со многими структурами данных и понимаю C++ на неплохом уровне.

Летняя школа Учёные Будущего

Мой первый опыт работы над большим проектом. Задание заключалось в том, чтобы создать прототип системы умного дома. Были поставлены следующие задачи:

  • С помощью микроконтроллеров, считать показания с датчиков внутри дома и отправить их на сервер.
  • Смоделировать и напечатать корпуса для устройств, чтобы они выглядели опрятно.
  • Разработать мобильное приложение для отображения данных и управления домом.
  • Создать сервер для принятия, хранения и получения данных.

Я занимался разработкой сервера, в ходе чего я работал...

  • С Mosquitto MQTT для получения сообщений от микроконтроллеров.
  • С Django для создания API для получения данных.
  • С PostgreSQL для создания базы данных.
  • С дистрибутивом Ubuntu, чтобы настроить удалённый сервер.

В итоге, наша команда заняла первое место, и мой серверный код сыграл в этом достижении немалую роль.

Второй год Яндекс.Лицея

Этот год обучения познакомил меня с промышленным программированием посредством трёх проектов:

  • Первый из них - приложение с графическим интерфейсом. Практика в использовании кроссплатформенного набора инструментов Qt и его языковой привязки PyQT5. Для сохранения данных пользователя, используется небольшая база данных SQLite. Также реализована работа с графиками посредством библиотеки matplotlib.
  • Второй - игра, созданная при помощи библиотеки pygame. Этот проект выполнялся в команде, и для обмена кодом использовалась площадка GitHub и система контроля версий Git.
  • Последний - веб-приложение, созданное при помощи фреймворка Flask. Для хранения данных использовалась техника ORM и база данных SQLite. Этот проект познакомил меня с веб-разработкой. Я работал с HTML-шаблонами и впервые использовал Bootstrap для стилизации.

И, конечно же, самый важный навык полученный в Яндекс.Лицее - умение искать информацию в интернете. Будь то документация какой-либо библиотеки или решение проблемы, я сумею найти и понять этот материал. В этом также помогает моё знание английского языка.
